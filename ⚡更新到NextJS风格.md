## ✨ 关键改进

### 从彩色卡片 → 纯文本链接
```

现在：what-is-heptabase... (纯文本)
```

### 从复杂装饰 → 极简设计
```
之前：背景色 + 圆角 + 阴影 + 编号徽章
现在：透明背景 + 纯文本 + 左边框高亮
```

```
之前：right: 0
现在：right: max(0, (100vw-1440px)/2)
      → 大屏居中对齐
```
